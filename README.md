## Add as a Dependency

### SBT (Scala 2.11 and 2.12)
```
"com.nthportal" %% "testing-utils" % "1.0.0" % Test
```

### Maven

**Scala 2.12**

```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>testing-utils_2.12</artifactId>
  <version>1.0.0</version>
  <scope>test</scope>
</dependency>
```

**Scala 2.11**

```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>testing-utils_2.11</artifactId>
  <version>1.0.0</version>
  <scope>test</scope>
</dependency>
```
